Peta Kode (index.html)

Komponen UI

  • Header/Toolbar: #themeBtn, #liteBtn, #langBtn, #accentBtn, #refreshBtn, search.
  • Sidebar: #profileCard, #statsCard, kontak & WA (#waModal).
  • Main: repos (#repoList, #repoStatus, #loadMoreBtn), About/Skills/Experience/Achievements.
  • Modal: #repoModal dan #waModal.

Data & Konstanta

  • GITHUB_USER, API_USER, API_REPOS, API_GISTS, API_ORGS
  • GH_HEADERS untuk GitHub REST API
  • ACCENTS palet aksen
  • I18N string UI

Konfigurasi

  • Username GitHub
const GITHUB_USER = "Renoslendra"; // ganti dengan username kamu
const API_USER  = `https://api.github.com/users/${GITHUB_USER}`;
const API_REPOS = `https://api.github.com/users/${GITHUB_USER}/repos?per_page=100&sort=updated`;
const API_GISTS = `https://api.github.com/users/${GITHUB_USER}/gists?per_page=10`;
const API_ORGS  = `https://api.github.com/users/${GITHUB_USER}/orgs`;
  • Nomor WhatsApp: ganti semua https://wa.me/6285156064912 dengan nomor kamu (format E.164 tanpa plus, contoh: 62812xxxx).
  • Aksen: edit array ACCENTS; default dikontrol --accent di :root.
  • Bahasa: tambah/ubah string di objek I18N.
  • SEO: perbarui meta dan OG tags (lihat bagian SEO).

Edit Konten (About/Skills/Experience/Achievements)

  • Target elemen: #aboutContent, #skillsContent, #expContent, #achContent

<script>
document.addEventListener('DOMContentLoaded', () => {
  document.getElementById('aboutContent').innerHTML = `

Cara dinamis (lanjutan)

async function loadSection(elId, url){
  const html = await fetch(url).then(r => r.text());
  document.getElementById(elId).innerHTML = html;
}
loadSection('aboutContent', 'https://gist.githubusercontent.com/<user>/<id>/raw/about.html');

Hindari menyisipkan script tak tepercaya via innerHTML.

Data Repositori & Statistik

  • Repos: diambil dari API_REPOS (max 100 per halaman, sort=updated).
  • Filter/sort di client; “Muat lebih banyak” menambah batch (atur ukuran batch di handler tombol).
  • Statistik total (bintang/fork/repos) dihitung agregat dari data repos yang dimuat.
  • Modal repo memuat detail (updated, created, default branch, dll.).

Tema, Aksen, Lite

  • Dark/Light: toggle class dark via tombol header.
  • Lite mode: class lite mengurangi blur/shadow.
  • Aksen: definisikan di ACCENTS, default via --accent.
  • Tokens: --radius, --shadow-*, --blur untuk perubahan global cepat.
  • Breakpoint utama: ~1080/1024/960 px.

I18N (ID/EN)

  • String UI ada di objek I18N; gunakan kunci yang sama di tiap bahasa.
  • Elemen memanfaatkan data-i18n-key/data-i18n-placeholder.
  • Toggle bahasa tersedia di header (persist di storage).
// Tambah kunci baru
I18N.id["hero.tagline"] = "Bangun cepat, tampil modern";
I18N.en["hero.tagline"] = "Build fast, look modern";

SEO & Metatag

  • Perbarui: description, keywords, author, theme-color.
  • OG: og:title, og:description, og:image, og:url.
  • Twitter Card: summary_large_image. Schema.org JSON‑LD Person.
<meta property="og:image" content="https://domainmu.com/og-image.jpg" />

Aksesibilitas

  • Gunakan ARIA pada kontrol baru; pertahankan role/aria-* di modal dan tombol.
  • Cek kontras setelah mengubah aksen/latar.
  • Hormati preferensi reduced motion.

Performa & Praktik Baik

  • Hindari set innerHTML berulang dalam loop besar.
  • Debounce input filter/sort jika perlu.
  • Opsional: cache fetch di localStorage, invalidasi via “Muat Ulang”.

Batasan & Rate Limit API GitHub

  • Tanpa autentikasi: ~60 request/jam per IP.
  • Hindari refresh berulang saat uji.
  • Jangan menaruh token di client; pakai backend/proxy untuk data private.

Deployment

  • GitHub Pages: Settings → Pages → Source main / root.
  • Atau host statis lain (Netlify, Vercel).
  • Custom domain: atur DNS + CNAME.
